Character Classes

Every adventurer has a class that defines their heroic capabilities. You-Meet-In-A-Tavern (YMIAT) adapts classes from Tales of the Valiant and the Black Flag Reference Document, compressing 20-level progression into 10 meaningful levels, using Fitness, Insight, and Willpower instead of traditional six abilities.

Subclass features arrive at 2nd, 4th, 6th, and 8th level. Improvement (talents or ability increases) arrives at 3rd, 5th, 7th, 9th, and 10th level. Casters use Spell Power and a max spell level by character level instead of daily spell slots. Each class lists one save with advantage—the class's primary saving throw. Progression features link to the class ability pages (one page per class).
